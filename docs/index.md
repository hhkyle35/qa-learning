## Quick Start

This knowledge base is built with [MkDocs](https://www.mkdocs.org/) and the Material theme.

To view locally:
```bash
mkdocs serve
```

To build for deployment:
```bash
mkdocs build
```

## How to Use

- **Search**: Use the search bar at the top to find content across all projects
- **Navigation**: Use the sidebar to browse by project or topic
- **Edit**: All content is in markdown files in the `docs/` directory
- **Version Control**: All files are tracked in Git for easy versioning
