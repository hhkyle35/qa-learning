# Templates

Useful templates for common documentation needs.

## Meeting Notes Template

```markdown
# Meeting Notes - [Date]

## Attendees
- Person 1
- Person 2

## Agenda
1. Topic 1
2. Topic 2

## Discussion
_Key points discussed_

## Decisions
- Decision 1
- Decision 2

## Action Items
- [ ] Action item 1 - Owner: Name
- [ ] Action item 2 - Owner: Name

## Next Steps
_What happens next_
```

## Project Update Template

```markdown
# Project Update - [Date]

## Progress
_What was accomplished_

## Blockers
_What's blocking progress_

## Next Week
_Planned work_

## Questions
_Questions that need answers_
```

## Learning Note Template

```markdown
# [Topic] - [Date]

## What I Learned
_Key takeaways_

- Resource 2

_Questions to explore further_
```
